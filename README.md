Polynomial Solver is an application for the Flipper Zero that enables users to solve and manipulate polynomials up to the sixth degree. With this app, users can evaluate polynomials, calculate derivatives and integrals, and visualize results directly on the Flipper's screen.

Features
Polynomial Solver: Input coefficients for a polynomial up to the sixth degree and evaluate its value for a user-defined 𝑥.
Derivative Calculation: Compute the derivative of the polynomial, reducing its degree.
Integration: Calculate the integral of the polynomial, increasing its degree (up to a maximum limit).
Adjustable 𝑥 Values: Allows users to set and modify the value of 𝑥 before solving the polynomial.
Scrollable Results: View long polynomial equations by scrolling horizontally on the Flipper screen.
Simple Navigation: Intuitive menus and input options using the Flipper Zero's buttons.
Getting Started
To install and use the Polynomial Solver app, follow these steps:

Clone or download this repository.
Build the app using uFBT (ensure you have the Flipper Zero SDK set up).
Transfer the .fap file to your Flipper Zero device.
Launch the app from the Flipper Zero's menu.
Controls
Main Menu:

Up: View instructions.
OK: Start setting 𝑥 value.
Back: Exit the app.
Set 𝑥 Value:

Up: Increase 𝑥.
Down: Decrease 𝑥.
OK: Confirm 𝑥 value and proceed to input coefficients.
Back: Cancel and return to the main menu.
Set Coefficients:

Up: Increase the coefficient value.
Down: Decrease the coefficient value.
OK: Move to the next coefficient.
Back: Return to the previous menu.
View Polynomial:

Left/Right: Scroll through long equations.
OK: Access calculus options (derivatives and integrals).
Back: Return to the main menu.
Calculus Options:

Up: Take the derivative.
Down: Take the integral.
Back: Return to the polynomial view.
